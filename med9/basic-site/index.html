The chipset of the newest MacBook

Apple ships the new Macbooks with very powerful custom made chips (made by Apple). These chipsets contain the followings:

  • CPU
  • GPU
  • RAM
  • Neural Engine
  • Secure Enclave
  • SSD controller
  • Image signal processor
  • Encode/decode engines
  • Thunderbolt controller with USB 4 support
  • ...
